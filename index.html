📖 社交值奕系统 · 使用指南

🧠 一、系统简介

社交值奕系统是一款集思维模型武器库、关系值评估、圈层闭环压力测试于一体的社交决策工具。101+个思维模型按四大领域分类，每个模型标注细分标签便于精准检索，帮助你从"凭感觉社交"升级为"用系统社交"。

🔍 发现问题→ 🧠 匹配模型→ 📊 评估现状→ 🔥 压力测试→ 🎯 行动闭环
🔐 二、账号与权限
1注册账号
首次打开系统，点击「首次使用？注册新账号」，输入用户ID（英文/数字）、显示昵称、密码（≥6位），点击注册。
💡 每个账号数据独立隔离，多人共用设备也互不干扰
2登录
输入注册时的用户ID和密码即可登录，系统自动恢复你上次的使用数据。
🎯 三、增强分析 — 智能分析引擎 + AI智囊

「增强分析」提供两种分析模式，顶部切换栏可自由切换：

🧠 模式一：智能分析引擎（离线可用）

1输入你的问题
在输入框中用一句话描述你当前遇到的问题。输入2个字以上即可看到实时联想推荐，覆盖1000+常见社交/职场/关系场景。
💡 也可以点击下方的热门提问标签，一键填入高频问题
2查看智能分析报告
点击「智能分析」后，引擎从1000+问题库 × 101+思维模型中精准匹配，生成结构化报告：问题诊断 → 武器库精准匹配(含匹配度) → 核心行动方案 → 跨领域组合洞察 → 误区提醒 → 相关问题推荐。
3深度探索
点击「你可能还想问」中的相关问题可继续深挖；点击底部按钮跳转RVS评估或压力测试，形成完整闭环。
4查看模型详情 & 跳转评估
点击模型卡片查看7段式详情；报告底部可一键跳转RVS评估或压力测试，形成闭环。

🤖 模式二：AI智囊（需联网+API Key）

1配置AI接口
点击顶部「⚙ API」按钮，选择AI服务商并填入API Key：
· DeepSeek（推荐，性价比高）→ 填入 sk- 开头的Key
· OpenAI → 填入Key，自动配置gpt-4o-mini
· 自定义端点 → 填写Base URL和模型名，兼容任何OpenAI格式API
💡 API Key仅保存在你的浏览器本地，不会上传到任何服务器
2开始对话
切换到AI智囊模式后，在输入框中描述你的问题，AI会自动检索系统内101+个思维模型中最相关的5个，作为上下文辅助分析，给出更深入、更有针对性的建议。
3持续追问
AI支持多轮对话，你可以针对建议继续追问、要求举例、请AI帮你制定行动计划。聊天记录按账号保存，下次打开可继续。
💡 AI回复中出现的【模型名】可点击跳转查看该模型详情
4离线降级
未配置API Key时，AI智囊模式会提示你先配置，规则分析模式始终可用。两种模式互不干扰，随时切换。

📊 五、RVS关系值评估 — 量化一段关系
1填写评估对象
在「📌 评估对象」输入框填写关系人/组织名称，如"张三"、"A公司客户"。名称会自动关联到评估报告中。
2逐维度打分（经典模式）
6个维度依次评估，每维度4题，使用1-6分量表（1=完全不符，6=完全符合）：
① 信任 — 这段关系的可靠性
② 互惠 — 双向价值交换的均衡度
③ 深度 — 情感与认知的连接深度
④ 支持 — 关键时刻的支持力度
⑤ 响应 — 沟通与反馈的及时性
⑥ 闭环 — 承诺→执行的完成度
💡 顶部进度条可点击跳转到任意维度，也可用「上一维度」「下一维度」按钮
3选择评估场景（动态题库）
评估对象下方有5个场景按钮：
· 🌐 通用 — 默认模式，适用于任何关系
· 🏢 职场 — 针对上下级、同事、客户关系
· 🤝 社交 — 针对朋友、社群、泛社交关系
· 💕 亲密 — 针对伴侣、家人、密友关系
· 🎲 随机 — 每次随机抽取一个场景，增加回访新鲜感
💡 切换场景后题目会自动更换为该场景的专属版本，同一关系可多次评估不同场景
4查看评估报告
完成全部24题后点击「生成完整报告」，系统生成：
· 百分制评分（原始总分24-144映射为0-100）
· 圈层等级（核心圈/紧密圈/协作圈/外围圈/弱连接）
· 雷达图（6维度可视化对比）
· 诊断报告（最弱维度+问题识别）
· 行动工具箱（针对性提升建议）
5保存记录与成长追踪
点击报告底部的「💾 保存评估记录」，本次评估结果会保存在你的账号数据中，可在「数据中心」查看历史记录。

📈 成长追踪：对同一关系做第2次以上评估时，报告底部自动显示「成长追踪」面板，6维度逐一对比上次分数（↑提升 / ↓下降 / →持平），总分变化趋势一目了然。同一关系不同场景的评估也会被追踪。
🧬 六、自适应评估 — 金字塔晋级系统

🔄 两种模式切换

在RVS评估或压力测试Tab顶部，有「🧬 自适应评估模式」开关。开启后隐藏经典评估UI，切换为自适应模式；关闭则恢复经典模式。两种模式互不干扰。

1基线测试定级
首次开启自适应模式，系统从白银(L2)难度出题，根据你的得分率确定起始层级。得分率≥80%可升级，<40%则降级，≥95%首次可跳级。
2金字塔可视化
5层金字塔实时展示你的当前层级：🥉青铜 → 🥈白银 → 🥇黄金 → 💎钻石 → 👑王者。每个层级下方显示已达到该层级的维度标签，当前层级有脉冲动画标识。
3通关指南
点击「📖 查看通关指南」，每个层级提供：核心提示、推荐思维模型（可点击跳转）、行动清单。帮你从当前层级突破到下一层。
4持续晋级
每次自适应测试8题，系统智能选择短板维度优先出题+随机混入高层探索题+已答去重。连续2次≥80%即升级，<40%降级。提交后查看层级变化和维度明细。
Q：经典模式和自适应模式的数据互通吗？
A：两种模式独立运行，经典模式评估24题出报告，自适应模式跟踪你的层级晋级进度，互不影响。
🔥 七、圈层闭环压力测试 — 检测关系压力源
1回答18题（经典模式）
3大维度×6子维度×3题/子维度，1-6分量表：
· 信息闭环压力 — 信息同质化 + 认知封闭程度
· 互惠失衡压力 — 单向消耗 + 隐性成本
· 结构弹性压力 — 退出壁垒 + 身份绑定
💡 顶部进度条实时显示答题进度
2选择测试场景
题目上方有4个场景按钮：
· 🌐 通用 — 默认模式，适用于任何圈层
· 🏢 职场 — 针对职场圈层的闭环压力
· 🤝 社交 — 针对社交圈层的闭环压力
· 💕 亲密 — 针对亲密关系的闭环压力
💡 切换场景后题目会自动加场景前缀，帮你代入具体情境
3查看深度压力报告
点击「⚡ 生成深度压力报告」，系统生成：
· 综合压力分（18-108分）和5级压力评级
· 6子维度精准画像（柱状图+文字解读）
· 压力溯源（最严重压力子维度+成因分析）
· 开环处方（针对性降压策略+推荐思维模型）
4一键跳转
报告底部的跳转按钮可快速进入RVS评估（量化该关系的健康度）或增强分析（匹配应对模型），形成完整闭环。
5保存记录与成长追踪
点击「💾 保存测试记录」，历史记录可在数据中心查看对比。

📈 成长追踪：第2次以上测试时，报告底部自动显示「成长追踪」面板，3维度（信息闭环/互惠失衡/结构弹性）逐一对比上次分数，总分变化趋势一目了然。压力分下降=改善，上升=加重。

❓ 十一、常见问题
Q：我的数据保存在哪里？安全吗？
A：所有数据保存在你浏览器本地的 localStorage 中，按账号ID隔离。不会上传到任何服务器。清除浏览器数据会导致记录丢失，建议定期在数据中心导出备份。
Q：换了手机/电脑能同步数据吗？
A：当前版本数据仅存于本地设备，不支持跨设备同步。你可以在数据中心导出数据文件，在新设备上导入恢复。
Q：忘记密码怎么办？
A：当前版本不支持密码找回（本地存储无后端），请牢记密码。如确实忘记，需注册新账号。
Q：自适应评估的层级会降级吗？
A：会。如果某维度连续测试得分率<40%，该维度会降一级。这是为了确保层级真实反映你的当前水平，而不是历史最高水平。
Q：别人打开我分享的链接能看到我的数据吗？
A：不能。数据按设备+域名隔离在 localStorage 中，别人打开链接看到的是空白登录页，需要自己注册账号。你的数据只存在于你的设备上。
Q：AI智囊的API Key安全吗？
A：API Key仅保存在你浏览器的 localStorage 中，按账号隔离，不会上传到任何服务器。清除浏览器数据会同时清除Key，建议妥善保管。
Q：AI智囊会收费吗？
A：本系统不收取任何AI费用。你使用的是自己配置的API Key，费用由AI服务商按其标准收取。DeepSeek价格极低（约0.001元/次），推荐使用。
Q：没有配置API Key能用AI智囊吗？
A：不能。AI智囊需要联网调用AI接口，未配置Key时会提示你先配置。🎯规则分析模式始终可用，不依赖网络。
Q：AI智囊和规则分析有什么区别？
A：规则分析是离线的，基于关键词匹配101+个思维模型生成固定格式报告，响应快；AI智囊是联网的，AI会检索相关模型知识后进行深度推理和多轮对话，建议更个性化。两者互补，推荐先用规则分析定位，再用AI智囊深挖。
Q：系统更新后我的数据会丢失吗？
A：不会。系统内置跨版本数据持久化机制：每次操作自动生成备份包，更新链接后用原账号登录即可自动恢复。如果自动恢复失败，可在登录页点击「🔧 恢复数据」输入恢复码或导入备份文件手动恢复。
Q：RVS评估的「场景」有什么用？
A：场景让评估题目更贴合你当前的关系情境。比如选「🏢职场」后，题目会围绕上下级、同事关系来问，比通用模式更有针对性。同一关系在不同场景下的评估结果可能差异很大，建议对重要关系做多场景评估。
Q：成长追踪是怎么工作的？
A：RVS评估对同一关系做第2次以上评估、压力测试做第2次以上时，报告底部自动出现「成长追踪」面板，对比各维度分数变化。RVS按同一关系名匹配，压力测试按全部历史对比。这样你可以看到自己的改善是否生效。

② 充值（10元可用很久，约0.001元/次对话）
③ 进入「API Keys」页面，点击「创建 API Key」
④ 复制 sk- 开头的字符串，粘贴到上方「API Key」输入框
⑤ 提供商选 DeepSeek，点击保存即可

① 打开 platform.openai.com
② 创建 API Key，粘贴到上方，提供商选 OpenAI
方式三：自定义接口
选择「自定义」，填写兼容 OpenAI 格式的 Base URL 和模型名即可
